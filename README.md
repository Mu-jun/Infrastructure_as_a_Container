# Infrastructure_as_a_Container
쿠버네티스와 도커 등을 활용하여 컨테이너 인프라 환경을 구축해보려 한다.
- 길벗출판사의 "컨테이너 인프라 환경 구축을 위한 쿠버네티스/도커"라는 책을 보면서 실습한 내용들이다
- 예제소스 : https://github.com/sysnet4admin/_Book_k8sInfra
